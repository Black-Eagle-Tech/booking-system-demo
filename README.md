# NovaCare — Booking & Appointment System Demo

Modern booking and appointment system built with Vue 3 + Vite + TailwindCSS v4.

## Features

- Modern SaaS UI
- Responsive layout
- Booking cards
- Appointment dashboard
- Dark mode aesthetic
- Vue Router structure
- Modular architecture
- TailwindCSS v4
- Ready for customization

---

## Tech Stack

- Vue 3
- Vite
- TailwindCSS v4
- Pinia
- Vue Router
- Lucide Icons

---

## Installation

Extract the ZIP and run:

```bash
npm install
```

---

## Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:5173
```

---

## Build Production

```bash
npm run build
```
